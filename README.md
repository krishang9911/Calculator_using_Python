# Python Calculator Project

## Description
A simple console-based calculator written in Python.  
Supports: **Addition**, **Subtraction**, **Multiplication**, and **Division**.  
Includes input validation and division-by-zero handling.

This project demonstrates:
- Building a small software project in **Python**
- Using **Visual Studio Code** for development
- Hosting a project on **GitHub**
- Creating proper repository structure
- Writing documentation in **Markdown**
